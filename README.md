# Oddurs Gummyworm

## How do I install these formulae?

`brew install oddurs/gummyworm/<formula>`

Or `brew tap oddurs/gummyworm` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "oddurs/gummyworm"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
